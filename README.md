# Telco Customer Churn Prediction

## Overview
[cite_start]This repository contains the coursework implementation for **CM2604: Machine Learning**. [cite_start]The goal of this project is to develop a binary classification model to predict whether a telecommunications customer will "churn" (cancel their service) based on various customer attributes.

## Dataset
The dataset used is the **Telco Customer Churn** dataset from Kaggle.
* [cite_start]**Source:** [Kaggle - Telco Customer Churn](https://www.kaggle.com/blastchar/telco-customer-churn) [cite: 19]
* [cite_start]**Attributes:** Includes customer demographics, account information, and service details (e.g., Contract type, Monthly Charges, Tenure)[cite: 18].
